Guía Rápida de Git

¿Qué es Git?

Git es un sistema de control de versiones distribuido, lo que significa que el historial completo del código está disponible en la computadora de cada desarrollador.

Comandos Básicos

  • git init: Inicializa un nuevo repositorio.
  • git add: Prepara los archivos para el commit (staging).
  • git commit: Guarda los cambios en el historial.
  • git status: Muestra el estado actual de los archivos.

Ramas en Git

Las ramas permiten trabajar en nuevas funciones sin afectar la versión principal del proyecto[cite: 9].

Preguntas Frecuentes

¿Para qué sirve el comando checkout? Sirve para cambiar de una rama a otra o para crear nuevas ramas si se usa el parámetro -b.
